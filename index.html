使い方ガイド

  1. この画面で身長・体重・頻度・経験レベル・目標を入力し「メニューを再生成」を押す
  2. Workout タブで今日のメニューを確認
  3. ジムで実際にトレーニングしながら、重量・回数を修正
  4. 各セットの ✓ をタップ → 完了マーク＋レストタイマー自動スタート
  5. セット数の増減やカスタム種目の追加も自由に可能
  6. 全種目終了後「記録を保存」を押す
  7. 次回のWorkoutでは、実績に基づいて重量が自動調整される

目標に応じて回数・重量・休憩時間が最適化されます。

目標回数重量休憩
筋力アップ3〜6回高重量（85%1RM）2〜3分
筋肥大8〜12回中重量（70%1RM）60〜90秒
持久力12〜20回低重量（55%1RM）30〜45秒

迷ったら「筋肥大（バランス型）」がおすすめ。細マッチョを目指すなら最も効率的です。

あなたの体重と経験レベルから、種目ごとの推定1RM（1回だけ挙げられる最大重量）を算出します。

推定1RM = 体重 × 種目別係数（レベルで変動）
作業重量 = 推定1RM × 目標別強度

この作業重量を各マシン/バーベルの刻み幅に丸めて提案します。回数・セット数・インターバルもすべて選択した目標に応じて自動設定されます。

経験レベルによって推定1RMの算出係数が変わります。レベルが上がるほど、より重い重量が提案されます。

レベル目安特徴
初心者0〜1年フォーム習得を重視。軽めの重量からスタート
中級者1〜3年筋肥大を本格的に狙う。中程度の重量
上級者3年以上高重量で限界に挑戦。体重比の係数が最も高い

例: ベンチプレス（体重70kgの場合）
初心者 → 約25kg / 中級者 → 約41kg / 上級者 → 約61kg

トレーニング記録を基に、次回の推奨重量を自動で調整します。2回目のWorkoutから発動します。

条件調整
全セットで目標回数を達成+2.5kg（重量UP）
1回だけ目標未達同じ重量を維持
2回連続で目標未達-2.5kg（デロード）

少しずつ重量を上げることで、筋力と筋量を着実に増やしていく「漸進性過負荷」の原則に基づいています。

週の頻度に応じて、最適な部位分割でメニューを構成します。

頻度分割法
週2回全身A / 全身B
週3回Push / Pull / Legs
週4回上半身A / 下半身A / 上半身B / 下半身B
週5回胸 / 背中 / 肩・腕 / 脚 / 弱点部位
週6回Push×2 / Pull×2 / Legs×2

頻度が多いほど各部位を細かく分けて集中的に鍛えられます。初心者は週3回（PPL）がおすすめです。

  • 継続が最重要 — 完璧なメニューより、続けることが成果を生みます
  • フォーム優先 — 重量を上げるより正しいフォームを意識しましょう
  • 睡眠と栄養 — 筋肉はトレーニング中ではなく回復中に成長します
  • タンパク質 — 体重×1.6〜2.0g/日を目安に摂取しましょう
  • 記録を見返す — Progressタブで成長を確認してモチベーション維持
